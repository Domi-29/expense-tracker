# 💰 Expense Tracker – Fullstack Application

A modern fullstack **Expense Tracker web application** built to manage personal finances efficiently.  
The project demonstrates full CRUD functionality, frontend-backend communication, and data visualization.

---

## 🚀 Overview

This application allows users to:

- Add new expenses with category, amount, and description
- Edit existing expenses
- Delete expenses
- View all expenses in a structured list
- Analyze spending habits through a dashboard chart
- Search and filter expenses

The goal of this project was to practice building a **real-world fullstack application** using a modern tech stack and clean architecture.

---

## 🧠 Key Features

### 💸 Expense Management
- Create, update, and delete expenses
- Structured categorization (e.g. Food, Transport, Shopping)

### 📊 Dashboard
- Visual representation of expenses by category
- Built using charts for better financial overview

### 🔍 Search & Filtering
- Quickly find specific expenses
- Filter by category or keyword

### 💾 Persistent Storage
- Data stored in SQLite database (backend)

---

## 🛠️ Tech Stack

### Backend
- Python
- FastAPI
- SQLite
- Pydantic
- CORS Middleware

### Frontend
- React (Vite)
- JavaScript (ES6+)
- Axios
- Recharts
- CSS

---

## 🏗️ Architecture

The project is structured as a **separated fullstack application**:

- Backend handles API, database logic, and business rules
- Frontend handles UI, state management, and user interaction
- Communication via REST API

## ⚙️ Installation & Setup

### 1️⃣ Clone repository

git clone https://github.com/Domi-29/expense-tracker.git
cd expense-tracker

BACKEND
cd expense-tracker-backend
python -m venv venv
source venv/bin/activate   # macOS / Linux
pip install -r requirements.txt
uvicorn main:app --reload

FRONTEND 
cd expense-tracker-frontend
npm install
npm run dev
